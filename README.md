# Smart Design Driving School — Management Dashboard

A complete operational dashboard for driving schools — students, lessons, 
driving tests, instructors, vehicles, progress tracking, payments, 
financial reconciliation, reporting, and a client-facing Share Demo feature.

## Quick Start
npm install
npm run dev

Opens automatically with sample data — no setup needed to preview.

## Deploy to GitHub
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPO.git
git branch -M main
git push -u origin main

## Deploy to Vercel
1. vercel.com → Add New → Project
2. Import your GitHub repo
3. Leave build settings as default
4. Click Deploy
